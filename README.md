# WeatherPage
simple weather page using OpenWeatherMap, C#, OSM and Angular. 


C# manda llamar la API de OWM y la expone con una minimal web API, Angular consume la API de C#.

Prueba de concepto simple.

Para instalar y usar, clonar el repositorio, abrir el servidor con un IDE correspondiente e instalar dependencias de NuGet; para el cliente hay que instalar los m√≥dulos de NPM y ejecutar ambos servicios.
